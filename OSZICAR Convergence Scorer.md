# OSZICAR Convergence Scorer

## Overview

The `oszicar_convergence_scorer` is a Python utility designed to parse and analyze the OSZICAR file generated by VASP (Vienna Ab initio Simulation Package). It evaluates the convergence behavior of electronic and ionic steps during a simulation.

## Features

- Parse OSZICAR files to extract energy and force convergence data.
- Analyze and score the convergence quality of VASP simulations.
- Provide insights into the stability and reliability of simulation results.

## Installation

1. Clone the repository:
    ```
    git clone https://github.com/your-repo/oszicar_convergence_scorer.git
    ```
2. Navigate to the project directory:
    ```
    cd oszicar_convergence_scorer
    ```
3. Install dependencies:
    ```
    pip install -r requirements.txt
    ```

## Usage

1. Place the `OSZICAR` file in the same directory as the script or specify its path.
2. Run the script:
    ```
    python oszicar_convergence_scorer.py --file path/to/OSZICAR
    ```
3. View the output, which includes convergence scores and analysis.
